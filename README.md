# NGO website 
 _The fullstack project with own CMS_
  
 _There is  authentication implemented on frontend and backend_
 
> [!IMPORTANT]
> To launch a website
> 1. npm install
> 2. configure .env files for frontend and backend
> 3. add credentials and connections for databases

## Stack used for a project
+ React
+ Express
+ Axios
+ JWT
+ Mongoose
+ MongoDB
+ Material UI
+ Multer (middleware for uploading files)
+ React-Quill
+ Bcrypt
+ Dompurify
